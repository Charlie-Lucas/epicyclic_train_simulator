# Simulateur d'Engrenage Épicycloïdal à Deux Étages

## Objectif

Dans le cadre d'impression d'un réducteur pour un moteur avec un réducteur 50:1, il fallait avoir précisément les configurations et dimensions possible d'un tel système.

## Description

Ce projet est un simulateur interactif permettant de modéliser et d'analyser le fonctionnement d'un réducteur épicycloïdal à deux étages avec satellites doubles. Il offre des visualisations dynamiques, des calculs précis des rapports de réduction, des couples et des vitesses, ainsi que des animations pour une meilleure compréhension des mécanismes en jeu.

## Fonctionnalités

Simulation du premier étage avec calcul automatique du nombre de dents des planètes.

Possibilité d'activer un second étage pour une analyse approfondie.

Calcul des rapports de réduction, couples et vitesses de sortie pour chaque étage.

Visualisation des tailles réelles des engrenages, incluant les diamètres primitifs et les entraxes.

Animation interactive illustrant le mouvement des engrenages en temps réel.

## Prérequis

Un navigateur web moderne (Chrome, Firefox, Edge, etc.).

Connexion internet pour charger la bibliothèque p5.js.

## Installation

Clonez ou téléchargez ce dépôt sur votre machine locale.

Ouvrez le fichier index.html dans votre navigateur.

## Utilisation

Réglez les paramètres des engrenages :

- Module

- Couple d'entrée (Nm)

- Vitesse d'entrée (tr/min)

- Nombre de dents du soleil

- Nombre de dents de la couronne extérieure (1er étage)

- Activation du second étage

- Nombre de dents des planètes (2e étage)

- Nombre de dents de la couronne extérieure (2e étage)

Visualisez les résultats et l'animation correspondante.

## Structure du Projet

- index.html : Contient la structure HTML et les éléments interactifs.

- script.js : Gère la logique de la simulation et les calculs associés.

- style.css : Fournit le style de base pour l'application.
